Aneka Cloud Computing

Aneka works on Application Platform as a Service for Cloud Computing. IT supports various programming models involving Task Programming, Thread Programming and MapReduce Programming and tools for rapid creation of applications and their seamless deployment on private or public Clouds to distribute applications. Characteristics of Aneka PaaS is to support provisioning of private cloud resources ranging from desktops, clusters to virtual datacenters using VMWare, Citrix Zen server and public cloud resources such as Windows Azure, Amazon EC2, and GoGrid Cloud Service.

Aneka technology primarily consists of two key components:

  1. SDK (Software Development Kit) containing Application Programming Interfaces (APIs) and tools essential for rapid development of applications which supports three popular Cloud programming models: 1.Task, 2. Thread & 3. MapReduce
  2. A Runtime Engine and Platform for managing deployment and execution of applications on private or public Clouds.

SUN FIRE X4470 Server

The laboratory houses a Sun Fire X4470 server powered by four Intel Xeon Processor 7500 Series eight core processor with hyper threading and with up to 512 GB of memory and over 1.8 TB of internal storage. It has almost double the memory capacity, more than quadruple the I/O subsystem bandwidth, and two-thirds more I/O expandability, making this the densest and fastest performing server in its class. The system is equipped with Sun Studio software with Open MP and MPI programming models. Xen Hypervisor is also loaded for experimental purposes in the area of Cloud computing.
